[IOS Firebase User Authentication Web App] Following Messages

INFO

It is a hybrid App developed on html and JavaScript and test on iOS device after development. It used the Firebase from Google Cloud service to provide the authentication function.


The idea of the app is to connect users' location, messages and the location (provided by the map services).


Wouldn't it be nice if users' can leave the messages in somewhere in a city, record or text whatever they want to say, and discovered by others when they walked through the place?


It can also be used as message platform for 2 users when they walk very closely in location.


The app is to experience mobile phone sensors, location services and messages.
